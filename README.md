This project was created as part of the CSS Grid Layout assignment. The objective was to recreate three modern UI designs using only HTML and CSS while focusing on CSS Grid concepts, positioning techniques, spacing, and layout structuring.

## 🎯 Assignment Objective

- Recreate any three provided designs as closely as possible while practicing:

- Build responsive layouts using CSS Grid.

- Practice grid-template-columns and grid-template-rows.

- Apply proper spacing and alignment techniques.

- Use relative and absolute positioning where required.

- Improve layout analysis and implementation skills.

## 🛠️ Technologies Used

- HTML5
- CSS3
- CSS Grid Layout

## 📚 CSS Concepts Practiced

- Flexbox Layout
- display: grid
- grid-template-columns
- grid-template-rows
- gap
- grid-area
- position: relative
- position: absolute
- Alignment with justify-content and align-items
- Responsive layout structuring
- UI Layering
- Spacing & Alignment
- Card Layout Design
- Typography Styling
- Image Positioning

## 🚀 Learning Outcomes

    Through this assignment, I learned how to:

- Analyze complex UI layouts.
- Convert designs into structured HTML.
- Create advanced layouts using CSS Grid.
- Manage spacing and alignment efficiently.
- Build reusable and maintainable CSS - structures.
- Improve frontend layout implementation skills.
